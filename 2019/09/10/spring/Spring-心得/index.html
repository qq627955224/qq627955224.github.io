常用Spirng注解

  • Configuration
  • Bean

一般情况下两者结合使用，当然也可以在Component对象中使用，但是@Configuration使用，可以做到管理

  • ComponentScan

指定扫描包路径，默认是当前类所在包以及子包，类似xml，<context:component-scan base-package=”com.acme”/>

  • Import

导入，如同xml ,导入多个配置文件，一般联合 Configuration使用，

如果ConfigB没有通过Import注解引入ConfigA,获取A会是空，通过增加注解import之后，可以正常引用

此注解的引入，诞生了一大批@Eanblexxx开头的注解

@import(class = xxxx) xxx 一般是 ImportSelector接口、获取@Configuration、ImportBeanDefinitionRegistrar

  • Condition

一个基于条件进行动态加载bean的注解，springboot进行充分扩展，多大10+，如：ConditionalOnClass、ConditionalOnBean、ConditionalOnJava等
