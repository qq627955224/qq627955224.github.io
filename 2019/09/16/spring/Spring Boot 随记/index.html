spring/Spring Boot 随记

事件

相信看过源码的同学应该都会发现，spring boot 在应用启动过程运用了大量的事件，事件按照spring boot的启动顺序进行发送，不同阶段对应不同事件，EventPublishingRunListener是整个spring boot 生命周期相关事件的发动者。事件主要有以下：

应用启动事件->环境准备事件->应用准备事件->应用完成事件

spring boot应用启动监听：SpringApplicationRunListener

SpringApplicationRunListeners listeners = getRunListeners(args);

listeners.starting();

​ run方法执行，发送启动事件

应用启动事件： ApplicationStartedEvent

相关监听：

LoggingApplicationListener
BackgroundPreinitializer
DelegatingApplicationListener
LiquibaseServiceLocatorApplicationListener

环境准备事件：ApplicationEnvironmentPreparedEvent

相关监听：

AnsiOutputApplicationListener
LoggingApplicationListener
BackgroundPreinitializer
ApplicationPidFileWriter
BootstrapApplicationListener
ConfigFileApplicationListener

ContextPrepared 准备事件

​ > 1.5版本发送空事件，2.0+发送ApplicationContextInitializedEvent

应用准备完成：ApplicationPreparedEvent

相关监听

LoggingApplicationListener
BackgroundPreinitializer
DelegatingApplicationListener
LiquibaseServiceLocatorApplicationListener

应用启动失败事件：ApplicationFailedEvent

相关监听

LoggingApplicationListener
BackgroundPreinitializer
DelegatingApplicationListener
LiquibaseServiceLocatorApplicationListener

应用就绪事件：ApplicationReadyEvent

相关监听

LoggingApplicationListener
BackgroundPreinitializer
DelegatingApplicationListener
LiquibaseServiceLocatorApplicationListener

笔记

ApplicationEvent

  • SpringApplicationEvent

    • ApplicationStartingEvent(启动事件)

    • ApplicationContextInitializedEvent (初始完成事件)

    • ApplicationEnvironmentPreparedEvent (环境、配置 预处理事件)

    • ApplicationPreparedEvent （应用预处理事件）

    • ApplicationStartedEvent (程序启动完成事件)

    • ApplicationReadyEvent (准备事件)

    • ApplicationFaildEvent（程序失败事件）

  • ExitCodeEvent

  • ParentContextAvaliableEvent

  • JobExecutionEvent

  • WebServerInitializedEvent

    • ReactiveWebServerInitializedEvent
    • ServletWebServerInitializedEvent

  • DataSourceSchemaCreatedEvent

  • PayloadApplicationEvent

  • ApplicaitonContextEvent

    • ContextClosedEvent
    • ContextRefreshedEvent
    • ContextStoppedEvent
    • ContextStartedEvent

初始化 EventPulishingRunLister,有SpringApplicationRunnListners 封装了 List

EventPublishingRunListener

  • starting -> ApplicationStartingEvent 事件
    • loggingApplicationListener
    • BackgroundPreinitializer
    • DelegatingApplicationListener
    • LiquibaseServiceLocatorApplicationListener
  • environmentPrepared -> ApplicationEnvironmentPreparedEvent
  • contextPrepared -> ApplicationContextInitializedEvent 事件
